# MTGCardBuilder

<!-- ![sample](./sample.gif) -->
<img src="./sample.gif" alt="sample" style="width: 800px;" />

基于vue制作的MTG卡牌生成器，可以通过填写各项参数来渲染出卡片效果。  
样式非完全依照实体卡样式，而是参考了MTGA内的卡牌样式。

## 运行步骤

``` bash
npm install
# 启动测试服务，访问localhost:8080
npm run dev
```
